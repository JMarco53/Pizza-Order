# Pizza-Order
This is my second project. tried to a bot to order me pizza. 
Make sure to install selenium module with this: pip install selenium
Also install your webrowser driver.

Chrome:	https://sites.google.com/a/chromium.org/chromedriver/downloads
Edge:	https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/
Firefox:	https://github.com/mozilla/geckodriver/releases
Safari:	https://webkit.org/blog/6900/webdriver-support-in-safari-10/

Check Selenium documentation for more help on installation: https://selenium-python.readthedocs.io/installation.html
